Start from scratch

If you choose start from scratch you will get options if you would like to choose stacks by your own or let platform choose for you.

If you decide to choose stacks by your own you will get a list from where you can choose your preferred stacks. If you let Platform decide for you it will redirect you to dashboard and will use stacks which best fits for your project.

Write Prompt

Once you are landed to dashboard you can start writing your Prompt.

Answer Questions

Based on your prompt Think4ever will ask some questions to understand your project requirement better. Answer them and Think4ever will start generating your project.

Suggestions

While you writting Prompt Think4Ever will suggest you few best prompt templates based on your project type. So you can use them directly on chat.

Generating Code

Once you answer all the questions Think4Ever will generate your code based on your answers. It will take some time to generate your code.
